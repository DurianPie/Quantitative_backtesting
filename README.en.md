# Quantitative_backtesting

#### Description
量化交易的回测系统主要功能为，读取股票历史数据，根据自定义的股票交
易策略（例如每隔一周，卖出当前持有的所有股票，并将当前资金分为 10 份，
分别买入一周天内跌幅最大的 10 支股票），在历史数据中进行模拟交易，得到最
终的收益率。
